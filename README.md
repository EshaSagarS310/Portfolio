# ✨ Esha Sagar – Developer Portfolio

  A modern, fully frontend-based developer portfolio built using HTML, CSS, JavaScript, and Bootstrap. It showcases personal details, skills, projects, experience, education, and certificates through a responsive and interactive UI. The site includes dynamic features like animations, modal project views, a lightbox for images, and a comment system powered by browser localStorage, with no backend or database involved. It is designed to be lightweight, fast, and easily deployable on static hosting platforms like GitHub Pages or Netlify, with a focus on clean UI, interactivity, and frontend development skills.

## 🌟 Overview

This portfolio is designed as a **real-world developer showcase project**, focusing on:

* ⚡ Fast performance (no backend dependencies)
* 📱 Fully responsive UI/UX design
* 🎨 Modern frontend aesthetics (glassmorphism + animations)
* 💾 Client-side persistence using `localStorage`
* 🧩 Modular UI sections with interactive components

It acts as both a **personal branding site** and a **frontend engineering demonstration project**.


## 🎯 Key Highlights

* 💻 100% frontend application (HTML, CSS, JS only)
* 🧠 Dynamic UI interactions using vanilla JavaScript
* 💬 Comment system using browser `localStorage` (no database)
* 🎞️ Smooth animations with AOS + Typed.js effects
* 🖼️ Image lightbox for projects & certificates
